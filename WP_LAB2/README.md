## Ответы на контрольные вопросы

### 1. Что делает тема в WordPress, а что — плагин?

**Тема** отвечает за внешний вид сайта: структуру страниц, расположение элементов, шрифты, цвета и общий дизайн. Тема определяет то, *как сайт выглядит*, но не влияет на его функциональность.

**Плагин** расширяет функциональность сайта — добавляет новые возможности, которых нет в стандартном WordPress. Например, Contact Form 7 добавляет формы обратной связи, Classic Editor меняет интерфейс редактора. Плагины работают независимо от темы и могут использоваться с любой темой.

### 2. Почему при смене темы контент сайта не теряется?

Весь контент сайта (записи, страницы, комментарии, медиафайлы) хранится в **базе данных MySQL**, а не в файлах темы. Тема лишь определяет способ отображения этих данных. При смене темы WordPress продолжает использовать ту же базу данных, поэтому весь контент остаётся нетронутым — меняется только визуальное оформление.

### 3. Как можно изменить внешний вид сайта без редактирования кода?

WordPress предоставляет несколько способов:

1. **Внешний вид → Настроить** — встроенный инструмент для настройки темы в реальном времени: изменение цветов, логотипа, заголовка, меню и виджетов.
2. **Смена темы** — установка новой темы из официального каталога полностью меняет дизайн без изменения контента.
3. **Виджеты** — в разделе Внешний вид → Виджеты можно добавлять готовые блоки контента в боковые панели и подвал сайта.
